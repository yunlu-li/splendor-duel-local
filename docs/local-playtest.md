# 本地试玩说明

## 启动

```bash
npm run dev
```

浏览器打开 Vite 输出的本地地址。

## 当前可玩内容

- 输入/修改 seed 后重开，可复现同一开局。
- 点击棋盘 token 选择，最多 3 个。
- 使用顶部按钮：
  - `拿选中 token`：执行强制拿 token。
  - `花特权拿选中`：选择 1 个非黄金 token 后执行可选特权行动。
  - `补充棋盘`：有空位且袋中有 token 时补盘。
- 市场卡下方按钮：
  - `购买`：使用规则引擎推荐的自动支付方案。
  - `保留`：自动拿棋盘上第一个黄金 token。
  - 牌堆区域 `盲保留`：保留对应等级牌堆顶牌。
- 玩家面板：
  - 显示 token、特权、威望、皇冠、bonus、同色分。
  - 保留区里的卡在可购买时可点击购买。
- Awaiting choice：
  - 弃 token。
  - 选择皇室卡。
  - 偷 token。
  - 拿同色 token。
  - Associate 复制 bonus。

## 原型限制

- 购买支付目前是自动推荐，暂未提供手动选择黄金抵扣分配。
- 保留卡自动选择棋盘上的第一个黄金，暂未让用户指定黄金位置。
- 页面没有隐藏对手保留卡，当前是本地调试视图。
- 还没有房间和 WebSocket；下一步会接入服务端房间。
